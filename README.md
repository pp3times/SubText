## Getting Started

First, run the development server:

1. Install package
```bash
npm run install
# or
yarn
# or
pnpm install
```

2. Run app
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
